# 二维码生成打印工具需求文档

## 1. 工具概述
### 1.1 工具名称
二维码生成打印工具

### 1.2 工具描述
专业的二维码生成和打印管理工具，支持批量处理、Excel数据导入导出，专为斑马打印机优化，适用于物料管理、库存标签等场景。

## 2. 核心功能
### 2.1 二维码生成功能
- 支持单个和批量二维码生成
- 根据物料信息自动生成二维码内容
- 支持多种二维码格式和尺寸设置

### 2.2 Excel数据管理
- **导入功能**：支持Excel表格批量导入物料数据
- **导出功能**：支持将生成的二维码信息导出为Excel格式
- **数据模板**：提供标准的Excel模板，包含料桶码、料品批次、托盘码、料品编码、规格型号、料品批次编码规则等字段

### 2.3 打印功能
- **斑马打印机支持**：专门适配斑马打印机打印协议
- **打印预览**：打印前可预览标签效果
- **明确打印提示**：每次打印操作都有清晰的状态提示和确认信息
- **批量打印**：支持选择多个项目进行批量打印

### 2.4 打印机配置管理
- **默认配置**：
  - 打印机类型：斑马打印机
  - 打印机名称：ZDesigner ZD888-203dpi ZPL
  - 打印机IP地址：172.16.5.199
- **配置修改**：支持用户自定义修改打印机设置
- **连接测试**：提供打印机连接状态检测功能

### 2.5 数据管理界面
- 物料信息列表展示，包含：原材来料编号、物料编号、物料名称、打印顺序、规格型号、托盘码、料桶码、料品状态、来料批次、供应商名等字段
- 支持数据筛选、排序和搜索功能
- 提供修改、删除、条码打印、导入、锁定批次、锁定料桶、恢复批次、恢复料桶、查询等操作按钮

## 4. 设计风格
### 4.1 配色方案
- 主色调：专业蓝色(#2E86C1)，体现工业化和专业性
- 辅助色：浅灰色(#F8F9FA)作为背景色，深灰色(#495057)作为文字色
- 功能按钮：绿色(#28A745)表示确认操作，橙色(#FD7E14)表示警告操作

### 4.2 视觉细节
- 圆角设计：按钮和卡片采用4px圆角，保持现代感
- 表格样式：清晰的边框线条，行间距适中便于数据查看
- 图标风格：简洁的线性图标，与打印和数据管理主题相关

### 4.3 整体布局
- 采用经典的工具型布局：顶部操作栏+主体数据表格+底部状态栏
- 左侧功能区域集中放置常用操作按钮
- 右侧显示打印机状态和配置信息
